# Começando

Vamos começar sua jornada no Rust! Há muito a aprender, mas toda jornada começa
em algum lugar. Neste capítulo, discutiremos:

- Instalando o Rust no Linux, macOS e Windows
- Escrevendo um programa que imprime `Hello, world!`
- Usando o `cargo`, o gerenciador de pacotes e sistema de build do Rust
