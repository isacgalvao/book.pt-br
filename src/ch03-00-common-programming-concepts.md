# Conceitos Comuns de Programação

Este capítulo aborda conceitos que aparecem em quase toda linguagem de
programação e como eles funcionam em Rust. Muitas linguagens de programação têm
muito em comum em sua base. Nenhum dos conceitos apresentados neste capítulo é
exclusivo de Rust, mas vamos discuti-los no contexto de Rust e explicar as
convenções para usá-los.

Especificamente, você aprenderá sobre variáveis, tipos básicos, funções,
comentários e controle de fluxo. Esses fundamentos estarão em todo programa em
Rust, e aprendê-los cedo dará a você uma base sólida para começar.

> #### Palavras-chave
>
> A linguagem Rust tem um conjunto de palavras-chave que são reservadas para uso
> exclusivo da linguagem, assim como em outras linguagens. Tenha em mente que
> você não
> pode usar essas palavras como nomes de variáveis ou funções. A maioria das
> palavras-chave tem significados especiais, e você as usará para realizar várias
> tarefas em seus programas Rust; algumas não possuem funcionalidade atualmente
> associada, mas foram reservadas para funcionalidades que possam ser
> adicionadas
> ao Rust no futuro. Você pode encontrar a lista de palavras-chave no [Apêndice
> A][appendix_a]<!-- ignore -->.
